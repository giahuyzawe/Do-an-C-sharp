FOOD TOUR APP - ANDROID APK
==========================

File APK chua duoc build. De build APK:

1. Mo Visual Studio
2. Mo project FoodStreetGuide
3. Chon Configuration: Release
4. Chon Target: Android -> apk
5. Build -> Publish
6. File APK se nam o: bin\Release\net8.0-android\publish\
7. Copy file APK vao thu muc nay va doi ten thanh: FoodTour.apk

Hoac dung command line:
   dotnet publish -f net8.0-android -c Release -p:AndroidPackageFormat=apk

Lien he dev neu can ho tro.
